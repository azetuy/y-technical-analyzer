# 株価パターン分析ツール 要件定義

## 概要

ティッカーコード（米国株）または証券コード（日本株）を入力すると、
Y式テクニカル分析に基づくローソク足パターン分析を表示するWebアプリ。

---

## 目的

- ローソク足パターンとMAの状態から今後の株価方向を判定
- 支持線・抵抗線を定量的に特定し、エントリーポイントを明確化
- 買鉄条件の自動チェックで感情を排除したトレード判断を支援
- リスクリワード比の事前計算で損小利大を実現

---

## 機能要件

### 1. 入力

| 項目 | 仕様 |
|------|------|
| 銘柄コード | 米国株ティッカー（例：AAPL）または日本株証券コード（例：7203） |
| 日本株 | 自動的に `{コード}.T` に変換（Yahoo Finance形式） |
| 時間軸 | 日足 / 週足 / 月足 切り替え |
| 表示期間 | 3ヶ月 / 6ヶ月 / 1年 / 2年 / 5年 |

### 2. ローソク足チャート

- インタラクティブなOHLCローソク足（Plotly）
- 出来高バーを下段に表示
- ズーム・パン操作対応

### 3. 移動平均線（Y式準拠）

| MA | 役割 |
|----|------|
| 5日MA | 短期トレンド。向きの変化が買鉄タイミングの鍵 |
| 20日MA | 中期基準。賞味期限の目安（20営業日） |
| 60日MA | ベースの重み。向きが変わりにくい信頼性高い線 |
| 400日MA | 20ヶ月線相当。長期トレンドの絶対的ベース |

### 4. Y式トレンド判定

**3フェーズ自動判定：**
- 🟢 **上昇トレンド**: 高値・安値が切り上がり、400日線上かつ上向き
- 🟡 **横ばい（レンジ）**: 高値安値が一定幅、60日MAが横向き
- 🔴 **下落トレンド**: 高値・安値が切り下がり、400日線下かつ下向き

**トレンドサイクル位置表示：**
```
上昇 → 横ばい → 下落 → 横ばい → 上昇
 ↑ 現在位置を表示
```

**パーフェクトオーダー判定：**
- 最強: 5MA > 20MA > 60MA（全て上向き）
- 監視: 5MA下向き、20MA・60MA上向き（押し目候補）
- エントリー: 5MAが上向きに転換し全て揃った瞬間
- 最弱: 全て下向き

### 5. 買鉄（かいてつ）パターン検出（Y式5条件）

最新ローソク足での5条件を自動チェック：

| # | 条件 | 判定 |
|---|------|------|
| 1 | 5MA > 20MA（並びが上昇を示す） | ✅/❌ |
| 2 | 20MAが上向き（上昇トレンド確認） | ✅/❌ |
| 3 | 5MAが下向き→上向きに変化 | ✅/❌ |
| 4 | ローソク実体が5MAの上に50%以上 | ✅/❌ |
| 5 | 株価が20MAに接近/タッチ（調整の証明） | ✅/❌ |

**買鉄強度**: 5条件中の達成数をスコアで表示
**損切り基準**: 5MA終値割れで機械的に切る → 自動表示

### 6. 高値安値ライン（Y式定義）

**定義（5MAの向き変化を区切りに使用）：**
- **高値**: 5MAが「下向き→上向き」変化区間の最高値（ヒゲ含む）
- **安値**: 5MAが「上向き→下向き」変化区間の最低値（ヒゲ含む）

**ライン格付け（当たり回数で強弱判定）：**
- 🔴 強い抵抗線（3回以上反応）
- 🟠 中程度の抵抗線（2回反応）
- 🔵 強い支持線（3回以上反応）
- 🟦 中程度の支持線（2回反応）

### 7. ギャップ（窓）検出

- ギャップアップ/ダウンを背景ハイライト
- **特に重要**: 上昇中の窓閉め（前のレンジ高値が抵抗になるパターン3）
- 窓の上限・下限を支持抵抗ラインとして表示

### 8. チャートパターン検出

| パターン | 説明 |
|----------|------|
| 毛抜き天井/底 | 同価格帯での複数ヒゲ（市場参加者が意識するライン） |
| 横ばい帯 | 3本以上のローソクが同価格帯に並ぶ状態 |
| カップ・ウィズ・ハンドル | U字型の底 + ハンドル（週足の買鉄形）|
| ダブルボトム | W字底（注意: Y式では「あまり強くない」） |
| 三角収束 | 高値が切り下がり安値が切り上がるパターン |

### 9. リスクリワード計算ツール

入力: エントリー価格・損切り価格・目標価格
出力:
- リスクリワード比（1:Xの形式）
- 目標値が1:5未満の場合は警告
- ポジションサイズ（許容損失額から逆算）

### 10. 市場地合い（ウォッチリスト）

以下の指標を簡易サイドバーに表示：

| 指標 | ティッカー | 用途 |
|------|-----------|------|
| ラッセル2000 | IWM | 炭鉱のカナリア |
| 恐怖指数 | ^VIX | 市場警戒感 |
| ビットコイン | BTC-USD | 大衆心理 |
| 米国債10年 | ^TNX | 金利動向 |
| 日経225 | ^N225 | 日本市場 |

## Y式の核心を実装に組み込む優先順位

1. **最優先**: 20ヶ月線（400日線）でのトレンド判定 → 全判断の土台
2. **高**: 買鉄5条件チェック → エントリータイミングの客観化
3. **高**: 高値安値ラインの自動抽出 → 支持抵抗の見える化
4. **中**: トレンドサイクル位置 → 次に起こることの絞り込み
5. **中**: ギャップ・毛抜き天井/底 → ライン強度の補強
6. **低**: カップ・ウィズ・ハンドル等 → より高度なパターン認識
